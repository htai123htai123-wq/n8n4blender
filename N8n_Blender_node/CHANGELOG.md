## [0.1.0] - 2023-12-XX

### 新增
- 初始版本发布
- 基本工作流编辑器界面
- 14种核心节点类型支持：
  - HTTP Request
  - Set
  - If
  - Switch
  - Merge
  - Code
  - Webhook
  - Manual Trigger
  - Schedule Trigger
  - Function
  - Function Item
  - No Operation
  - Start
  - Stop and Error
- 节点属性编辑面板
- 工作流导入/导出功能
- 多语言支持（中文简体、中文繁体、英文）
- 插件偏好设置
- 快捷键支持
- API连接配置
- 工作流执行状态监控

### 技术细节
- 基于Blender 4.2+ API开发
- 使用模块化架构设计
- 支持插件热重载
- 完整的错误处理机制

### 已知问题
- 工作流执行时可能出现延迟
- 大型工作流可能导致界面卡顿
- 部分高级节点配置尚未完全实现

## 版本说明

### 版本号规则
本项目使用语义化版本号：`主版本号.次版本号.修订号`

- **主版本号**：当进行不兼容的API修改时
- **次版本号**：当添加功能时，保持向后兼容
- **修订号**：当进行向后兼容的修复时

### 更新类型
- `新增` - 新功能
- `更改` - 对现有功能的更改
- `弃用` - 即将删除的功能
- `移除` - 已删除的功能
- `修复` - 错误修复
- `安全` - 安全相关的修复
